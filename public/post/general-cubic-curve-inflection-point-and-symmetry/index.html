1 Problem

Problem 40, page 157 in Simmons (2016)

Consider the general cubic equation

\[ \begin{equation} y = a x^3 + b x^2 + c x + d \tag{1.1} \end{equation} \]

Show that:

  1. It has one and only one point of inflection at \(I (\frac{-b}{3a}, k)\) where \(k = \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d\),
  2. It has one maximum point and one minimum point if and only if \(b^2 - 3ac \gt 0\),
  3. The abscissa of \(I\) is the average of the abscissas of the maximum and minimum point, and
  4. The curve is symmetric with respect to \(I\) by
    1. Introducing a new \(X\)-axis and \(Y\)-axis such that \(X = x + \frac{b}{3a}\) and \(Y = y - k\) and making \(I\) as the origin of the \(XY\)-system,
    2. Showing that the equation of the curve in the \(XY\)-system is \(Y = aX \left(X^2 - \frac{b^2 - 3ac}{3 a^2} \right)\), and
    3. Showing that this transformed equation is symmetric with respect to \(I\).

2 Solution

2.1 Point of Inflection

The point of inflection, at which the curve changes its concavity, of a curve can be determined by finding the root of the second derivative (\(f'' \left( x \right) = 0\)).

Computing for the second derivative of (1.1),

\[ \begin{align} y &= f \left( x \right) = a x^3 + b x^2 + c x + d \\ \\ f' \left( x \right) &= 3a x^2 + 2b x + c \tag{2.1} \\ \\ f'' \left(x \right) &= 6a x + 2b \tag{2.2} \end{align} \]

Finding the roots of (2.2),

\[ \begin{align} 6a x + 2b &= 0 \\ 6a x &= -2b \\ x &= \frac{-2b}{6a} \\ x &= \frac{-b}{3a} \tag{2.3} \end{align} \]

To find the ordinate of the point of inflection, substitute (2.3) to (1.1)

\[ \begin{align} y &= a \left( \frac{-b}{3a} \right)^3 + b \left( \frac{-b}{3a} \right)^2 + c \left( \frac{-b}{3a} \right) + d \\ \\ y &= a \left( \frac{-b^3}{27 a^3} \right) + b \left( \frac{b^2}{9 a^2} \right) + \left( \frac{-bc}{3a} \right) + d \\ \\ y &= \frac{-b^3}{27 a^2} + \frac{b^3}{9 a^2} - \frac{bc}{3a} + d \\ \\ y &= \frac{-b^3 + 3 b^3}{27 a^2} - \frac{bc}{3a} + d \\ \\ y &= \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d \tag{2.4} \end{align} \]

This proves that the one and only point of inflection of (1.1) is at \(I (\cfrac{-b}{3a}, \cfrac{2 b^3}{27 a^2} - \cfrac{bc}{3a} + d)\).

2.2 Maximum and Minimum

The extrema of a curve can be determined by finding the roots of the first derivative.

Equating (2.1) to zero,

\[ \begin{equation} 3a x^2 + 2b x + c = 0 \tag{2.5} \end{equation} \]

Before we solve (2.5), I think this is a good opportunity to review the quadratic formula. Given a generalized quadratic equation

\[ \begin{equation} a x^2 + b x + c = 0 \tag{2.6} \end{equation} \]

We can solve the roots of (2.6) with the quadratic formula

\[ \begin{equation} x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \tag{2.7} \end{equation} \]

The roots of (2.5) can be calculated using (2.7). However, we must be mindful of our coefficients.

\[ \begin{align} x &= \frac{-2b \pm \sqrt{\left( 2b \right)^2 - 4 \left( 3a \right) c}}{2 \left( 3a \right)} \\ \\ &= \frac{-2b \pm \sqrt{4b^2 - 12a c}}{6a} \\ \\ &= \frac{-2b \pm 2 \sqrt{b^2 - 3a c}}{6a} \\ \\ x &= \frac{-b \pm \sqrt{b^2 - 3a c}}{3a} \\ \\ x_{min} &= \frac{-b - \sqrt{b^2 - 3a c}}{3a} \tag{2.8} \\ \\ x_{max} &= \frac{-b + \sqrt{b^2 - 3a c}}{3a} \tag{2.9} \end{align} \]

Note that (2.8) and (2.9) will only exist if \(b^2 - 3ac \geq 0\).

2.3 Average of the Extrema

We can now determine if the abscissa of \(I\) is equal to the average of \(x_{min}\) and \(x_{max}\).

\[ \begin{align} \frac{x_{min} + x_{max}}{2} &= \frac{\cfrac{-b - \sqrt{b^2 - 3a c}}{3a} + \cfrac{-b + \sqrt{b^2 - 3a c}}{3a}}{2} \\ \\ &= \frac{\cfrac{-2b}{3a}}{2} \\ \\ &= \frac{-b}{3a} \tag{2.10} \end{align} \]

The abscissa of \(I\) is \(\cfrac{-b}{3a}\), which is also (2.10), the average of \(x_{min}\) and \(x_{max}\).

2.4 Symmetry and the XY-System

To convert the (1.1) in terms of the \(XY\)-system, change \(x\) and \(y\) in terms of \(X\) and \(Y\). From the given.

\[ \begin{align} x &= X - \frac{b}{3a} \\ \\ y &= Y + k = Y + \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d \\ \\ y &= a x^3 + b x^2 + c x + d \\ \\ \left( Y + \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d \right) &= a \left( X - \frac{b}{3a} \right)^3 + b \left( X - \frac{b}{3a} \right)^2 + c \left( X - \frac{b}{3a} \right) + d \tag{2.11} \end{align} \]

Before I proceed, I think this is a good opportunity to review two, in my opinion, must-know binomial expansions.

\[ \begin{align} \left( m + n \right)^2 &= m^2 + 2mn + n^2 \tag{2.12} \\ \\ \left( m + n \right)^3 &= m^3 + 3m^2n + 3mn^2 + n^3 \tag{2.13} \end{align} \]

We can use our knowledge of (2.12) and (2.13) to simplify (2.11).

\[ \begin{align} \left( X - \frac{b}{3a} \right)^3 &= X^3 - \frac{b}{a} X^2 + \frac{b^2}{3 a^2} X - \frac{b^3}{27 a^3} \\ \\ \left( X - \frac{b}{3a} \right)^2 &= X^2 - \frac{2b}{3a} X + \frac{b^2}{9a^2} \\ \\ Y + \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d &= a \left( X^3 - \frac{b}{a} X^2 + \frac{b^2}{3 a^2} X - \frac{b^3}{27 a^3} \right) + b \left( X^2 - \frac{2b}{3a} X + \frac{b^2}{9a^2} \right) + c \left( X - \frac{b}{3a} \right) + d \\ \\ Y + \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d &= a X^3 - b X^2 + \frac{b^2}{3 a} X - \frac{b^3}{27 a^2} + b X^2 - \frac{2b^2}{3a} X + \frac{b^3}{9a^2} + c X - \frac{bc}{3a} + d \\ \\ Y + \frac{2 b^3}{27 a^2} - \frac{bc}{3a} + d &= a X^3 - \frac{b^2}{3 a} X + \frac{2b^3}{27 a^2} - \frac{bc}{3a} + c X + d \\ \\ Y &= a X^3 - \frac{b^2}{3 a} X + c X \\ \\ Y &= a X^3 + \frac{-b^2X + 3 a c X}{3a} \\ \\ Y &= a X \left( X^2 - \frac{b^2 - 3 a c}{3 a^2} \right) \tag{2.14} \end{align} \]

Let’s review how to determine a curve’s symmetry. A curve defined by \(f \left( x \right)\) is symmetric with respect to the \(x\)-axis if \(f \left( x \right) = - f \left( x \right)\), with respect to the \(y\)-axis if \(f \left( x \right) = f \left( -x \right)\), with respect to the origin if \(f \left( x \right) = - f \left( -x \right)\). Thus, we can prove that (2.14) is symmetric to its origin, if \(Y = F \left( X \right) = - F \left( -X \right)\).

\[ \begin{align} -F\left( -X \right) &= - \left[ a \left\{-X\right\} \left( \left\{-X\right\}^2 - \frac{b^2 - 3 a c}{3 a^2} \right) \right] \\ \\ &= a X \left( X^2 - \frac{b^2 - 3 a c}{3 a^2} \right) \\ \\ &= F \left( X \right) \end{align} \]

We have proven that (2.14) is symmetric with respect to its origin.

3 Plot of the Generic Cubic Curve

I think we can look at how a generic cubic curve looks like with the three points (inflection, minimum, and maximum) included.

# Attach tidyverse

library(tidyverse)

# Set coefficients, chosen via trial and error, note that b^2 - 3ac > 0

a_coeff <- 0.3

b_coeff <- 2.2

c_coeff <- -0.9

d_coeff <- 17

# Calculate the point of inflection

inflection_x <- 
  - b_coeff / (3 * a_coeff)

inflection_y <- 
  (2 * b_coeff^3) / (27 * a_coeff^2) - 
  (b_coeff * c_coeff) / (3 * a_coeff) +
  d_coeff

# Calculate the maximum and minimum points

max_x <- 
  (-b_coeff - 
     sqrt(
       b_coeff^2 - 
         3 * a_coeff * c_coeff
     )) / (3 * a_coeff)

min_x <- 
  (-b_coeff + 
     sqrt(
       b_coeff^2 - 
         3 * a_coeff * c_coeff
     )) / (3 * a_coeff)

max_y <- 
  a_coeff * max_x^3 +
  b_coeff * max_x^2 +
  c_coeff * max_x +
  d_coeff

min_y <- 
  a_coeff * min_x^3 +
  b_coeff * min_x^2 +
  c_coeff * min_x +
  d_coeff

# Plot the general cubic equation with

tibble(
  abscissa = seq(
    inflection_x - 6, 
    inflection_x + 6, 
    by = 0.1
  ),
  ordinate = 
    a_coeff * abscissa^3 +
    b_coeff * abscissa^2 +
    c_coeff * abscissa +
    d_coeff
) %>% 
  ggplot(
    aes(
      abscissa,
      ordinate
    )
  ) +
  geom_line() +
  annotate(
    "point",
    x = inflection_x,
    y = inflection_y,
    color = "green"
  ) +
  annotate(
    "point",
    x = max_x,
    y = max_y,
    color = "red"
  ) +
  annotate(
    "point",
    x = min_x,
    y = min_y,
    color = "blue"
  ) +
  geom_hline(yintercept = 0) +
  geom_vline(xintercept = 0) +
  theme_minimal() +
  labs(
    x = "x",
    y = "y"
  )

Figure 3.1: The Generic Cubic Curve. The point of inflection is colored green, the minimum point blue, and the maximum point red.

Figure 3.1 shows that the extrema we computed are not absolute minimum and and maximum points of the curve.
