Drawing a Sierpinski Triangle in R

2019 Nov 28
for / gganimate / ggplot2

knitr::opts_chunk$set( message = FALSE, warning = FALSE, echo = TRUE, fig.align = "center", gganimate = list( width = 672, height = 480 ) ) Chaos Game I learned about this chaos game to draw a Sierpinski triangle. Select a random point within a triangle. Select a random vertex. Move to the midpoint of that point and the vertex. From the new point, repeat 1. Keep doing 1 to 4 until you draw a Sierpinski trianle.
